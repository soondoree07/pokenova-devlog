## 4. 주요 기능

### 4-1. 메인 페이지 (index.html)

- **Hero 섹션:** 최신 연도 썸네일 이미지 표시
- **About 섹션:** 작가 소개글 (bio), CV(약력) 블록
- **Works 섹션:** 연도별 카드 목록 — 클릭 시 해당 연도 첫 작품 라이트박스 바로 오픈
  - 연도 클릭 → 갤러리 그리드 단계 생략, 첫 번째 작품 라이트박스 즉시 열기
- **Contact 섹션:** 이메일, Instagram 링크

### 4-2. 연도 상세 페이지 (year.html)

- **갤러리 레이아웃:** CSS Grid 4열 (반응형: 3열/2열/1열)
  - 연도 페이지 진입 시 갤러리 그리드·히어로·nav·back-btn·footer 숨김 → 라이트박스만 표시
- **작품 크기 비례 레이아웃:** 물리적 cm → 화면 비례 (1cm = 6px, 320~1050px 범위)
- **이미지 캡션:** 각 작품 이미지 아래 제목 표시 (serif 폰트, 회색, 중앙 정렬)
  - `.masonry-item-img` 래퍼로 호버 줌 유지하면서 캡션 외부 노출

### 4-3. 라이트박스

- 키보드 ← → ESC 지원
- 이미지 영역: max-width 1400px, 90vh
- **라이트박스 X 버튼 닫기** → 메인 페이지의 해당 연도 클릭 위치로 복귀 (스크롤 위치 보존)
- 이미지 아래 작품 제목·재료·크기·설명 표시

### 4-4. 반응형 / 모바일

- 갤러리 4열 → 3열 → 2열 → 1열 반응형
- 모바일 햄버거 메뉴 (UX 개선, 아이템 컷오프 수정)
- iOS 자동 줌 방지: 입력 font-size 16px 이상 고정
- 터치 타겟 최소 44px
- 라이트박스 모바일 이미지 크롭 수정

## 5. 데이터 구조 (data.json)

```json
{
  "artist": {
    "name": "배정혜",
    "tagline": "한 줄 소개",
    "bio": "소개글",
    "email": "jh0329ms@naver.com",
    "instagram": "https://www.instagram.com/artist_jbae/"
  },
  "years": [
    {
      "year": 2026,
      "description": "연도 설명",
      "thumbnail": "images/2026/thumb.jpg",
      "works": [
        {
          "title": "작품 제목",
          "material": "Mixed Media",
          "size": "114×162cm",
          "description": "작품 설명",
          "image": "images/2026/파일명.jpg",
          "solo": true
        }
      ]
    }
  ]
}
```

**작품 필드:**
| 필드 | 설명 |
|---|---|
| `title` | 작품 제목 |
| `material` | 재료 (예: Mixed Media, Oil on Canvas) |
| `size` | 크기 (세로×가로cm) |
| `description` | 작품 설명 |
| `image` | 이미지 경로 (images/연도/파일명.jpg) |
| `solo` | 단독 배치 여부 (true: 1열 전체, 없으면 그룹 배치) |

## 7. 이미지 처리 원칙

- 흰색 테두리·캡션 자동 크롭 (Python 스크립트 처리)
- 이미지 압축 후 업로드
- `images_backup/` 폴더에 원본 백업 보관
- 크기(cm) 기재 시 세로×가로 순서 확인 필수 (오기재 이력 있음 — 픽셀 비율로 자동 검증)
- 작품 제목 대소문자: 영문 제목은 모든 단어 첫 글자 대문자 (Title Case)
- 줄바꿈 표현 필요 시 `\n` 사용 (렌더링 지원)
