## 1. 프로젝트 개요

박정혁 개인 포트폴리오 랜딩 페이지.
학부생 UX·콘텐츠 기획자 / 게임 기획자 지망생 정체성을
시네마틱한 다크 우주 테마로 정돈한 **1페이지 프리미엄 포트폴리오**다.

템플릿 DNA는 "Orbis.Nft" 스타일(liquid-glass UI + 비디오 배경 + Anton/Condiment
타이포)을 유지하되, NFT 톤은 완전히 제거하고 "야심 있는 디자이너·기획자"의
톤으로 재해석했다.

### 핵심 포지셔닝
- "빠르게가 아니라 **정확하게** 판단하는 게임 기획자로 성장한다"는 비전
- UX 플로우 · 시스템 설계 · 콘텐츠 기획 · 몰입형 미디어를 연결하는 학부 디자이너

## 2. 기술 스택

| 항목 | 내용 |
|---|---|
| 프레임워크 | React 18 + TypeScript |
| 빌드 | Vite 5 |
| 스타일 | Tailwind CSS 3 + 커스텀 liquid-glass CSS |
| 아이콘 | lucide-react |
| 폰트 | Anton (headings) · Condiment (accent) · system monospace (body) |

> 외부 패키지는 최소화. React + Vite + Tailwind + lucide-react 외 추가 의존성 없음.

## 5. 디자인 시스템

### 색상
- Background: `#010828` (딥 네이비)
- cream: `#EFF4FF` (기본 텍스트)
- neon: `#6FFF00` (Condiment accent / underline bar)

### 타이포그래피
- `font-grotesk` = Anton (모든 헤딩·네비·오버레이 라벨)
- `font-condiment` = Condiment (필기체 accent만)
- `font-mono` = system monospace (본문·라벨·값)

### Liquid Glass
- `background: rgba(255,255,255,0.01)` + `backdrop-filter: blur(4px)`
- `background-blend-mode: luminosity` + `box-shadow inset 0 1px 1px rgba(255,255,255,0.1)`
- `::before` 로 부드러운 상·하 그라데이션 테두리 마스킹
- 적용처: navbar · 아이콘 버튼 · 프로젝트 카드 · 오버레이 바 · CTA 아이콘 스택

## 6. 데이터 분리 원칙 (CRITICAL)

모든 콘텐츠는 **`src/data/*.ts`에서만** 관리한다.
컴포넌트에는 어떤 프로젝트명·카피·링크도 하드코딩하지 않는다.

### Project interface
```ts
interface Project {
  id: string;
  title: string;
  subtitle?: string;
  year: string;
  type: 'industry' | 'personal' | 'client'
      | 'academic' | 'capstone' | 'tool';
  featured: boolean;          // true → 3카드 그리드 자동 포함
  overlayLabel?: string;
  overlayValue?: string;
  videoUrl?: string;
  description: string;
  role?: string;
  stack?: string[];
  url?: string;
  repo?: string;
}
```

### 신규 프로젝트 추가 워크플로우
1. `src/data/projects.ts` 열기
2. 배열 최상단에 새 `Project` 객체 추가
3. (선택) `featured: true` — 대신 기존 카드 하나 `featured: false`
4. 저장. 컴포넌트 수정 불필요.

### 프로필·비전·연락처 변경 워크플로우
1. `src/data/profile.ts` 열기
2. 해당 문자열 수정
3. 저장.
