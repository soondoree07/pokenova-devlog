## 3. 핵심 기능

### 3.1 파티 관리
- 새 파티 생성 (이름 + 닉네임 6칸 그리드, Enter로 다음 칸 이동, 빈 칸은 무시, 1명 이상 입력 시 저장)
- 파티 목록 조회 / 카드 클릭으로 파티 상세 진입
- **파티 카드 우상단 × 버튼**으로 삭제 (호버 시 노출, 모바일은 옅게 항상 노출, confirm으로 기록 N건도 함께 삭제 안내)
- 파티 상세 우상단 "삭제" 버튼도 동일 동작

### 3.2 캘린더 뷰
- 월 단위 그리드, **메이플 주차 시작 (목→수)** — 요일 헤더: 목/금/토/일/월/화/수
- 토·일은 빨간색 강조 (`--accent-rose`)
- 오늘 날짜 골드 강조
- ←/→ 버튼으로 월 이동 + **모달 닫고 재렌더해도 보고 있던 월 보존** (`calendarViewByParty` Map)
- 셀 클릭 → 그날 모달
- 셀에 보스별 색상 pill (최대 4개 + `+N` 표시)

### 3.3 진행도 위젯
- **이번 주 현황** — 주간 7보스 chip, 클리어한 보스만 색상 활성화 (`color-mix`)
- **이번 주 1인 분배액** — 결정석 합 ÷ 파티원 수, 옆에 결정석 이미지(`png/결정석.webp`) 클릭 시 `#/crystals`로 이동
- **이번 달 현황** — 검마 chip + 1인 분배액 + 월간 결정석 이미지(`png/월간보스결정석.webp`)
- 분배액 합산은 **`cycle === 'weekly'` / `cycle === 'monthly'`만**으로 가드 (월간 검마가 같은 주에 잡혀도 주간 합에 안 더해짐)

### 3.4 회차 기록 흐름

**날짜 모달 (캘린더 셀 클릭)**
- 채널 picker 버튼 그리드 (1 / 20세이상 / 2 / 3 / ... / 39, 8열, 드롭다운 X)
- 회차가 1건이라도 있으면 그 채널로 잠김
- 회차 카드 N개 (보스 / 채널 / 상자 연 사람 / 기본 보상 / 전리품 목록 + 삭제 ×)
- "+ 보스 기록 추가" 버튼 — 채널 선택 전엔 비활성화

**기록 추가 폼**
- 보스 select — **같은 주 클리어한 주간 보스 / 같은 달 클리어한 검마는 자동 제외**
- 상자 연 사람 select / 기본 보상 select (form-row 2열) — 직전 회차의 값으로 자동 미리 채움
- 전리품 입력은 **타일 그리드** (이미지 + 이름, 토글 선택). select 폐기.
- 선택된 전리품마다 [이름 라벨 — 그룹/항목 색] [먹은 사람] [가격] [×] 행으로 자동 추가
- **저장 필수: 보스 + 상자 연 사람.** 기본 보상 / 전리품은 비워도 OK.

### 3.5 결정석 가격 편집 페이지 (`#/crystals`)
- 진행도 위젯의 결정석 이미지 클릭 시 진입
- 주간 보스 8종 (유피테르 포함) → 가로 2열 카드
- 월간 보스 검마 → 별도 섹션 카드
- 카드 좌측 보더 = 보스 색
- 페이지 하단 가운데 저장 버튼 1개 (한 번에 모든 가격 저장 → `crystalOverrides` 맵에 보관)
- 빈/0/숫자 아닌 값은 자동으로 기본값(BOSSES.crystal)으로 복귀

### 3.6 백업/복원
- `↓ 백업` 버튼 — 전체 데이터 JSON 다운로드 (`maple-boss-backup-YYYY-MM-DD.json`)
- `↑ 복원` 버튼 — JSON 업로드, confirm 후 덮어쓰기
- `crystalOverrides`도 백업/복원에 포함

```js
  ],

    [bossId]: number(억)
  }
}
```
