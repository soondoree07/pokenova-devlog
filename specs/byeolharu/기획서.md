## 1. 제품 요약

**한 줄 요약:** 후회를 성장의 연료로 변환하는 심리학 기반 개인 성장 모바일 앱.

**핵심 가치 제안:** "후회를 변환하는 가장 아름다운 1분, 나만의 밤하늘을 만들어간다."

유저는 자신의 후회를 기록하고 다짐으로 전환한 뒤, 극복 과정을 **별과 별자리**라는 시각적 메타포로 경험한다. 극복한 후회들이 모여 **나만의 밤하늘**을 완성한다.

### 제품 철학 두 축

1. **Transform, not Record** — 기록보다 변환. 일기 앱이 아님
2. **Regret-Positive Framing** — 후회는 "더 나은 선택지를 안다는 증거"

## 3. 핵심 기능

### 3.1 기록 3단계 플로우 (1분 이내)

| 단계 | 액션 | 설명 |
|---|---|---|
| 1 | 카테고리 선택 | 6개 중 택1 |
| 2 | 후회 한 줄 입력 | 희미한 별 생성 애니메이션 |
| 3 | 다짐 + 리뷰 날짜 + 별 이름(선택, 1~8자) | "또 같은 일이 일어나면 어떻게 할래?" |

**설계 원칙:** 감정 기록 단계 의도적 생략 (단순함 우선)

### 3.2 극복 선언 플로우 (2단계 분리)

```
[희미한 별 탭]
  ↓ [극복했어요]
[✨ 풀컬러 모핑 + 효과음 + 햅틱]  ← 즉시 보상 (0.8~1.2초)
  ↓ (1~2초 여운)
[부드러운 입력창 페이드인]
  → "이 극복의 순간을 기록해볼까요?"
  ├─ 스토리 입력 → 저장 + 아카이브
  └─ "나중에 할게요" → 저장 (스토리 미작성, 빛나는 별 재탭 시 1회 재프롬프트)
```

**설계 근거:** 변환 경험이 입력 장벽보다 우선. 몰입 최고조 유지.

### 3.3 리뷰 날짜 도래

푸시 알림 → 앱 진입 → 해당 별 하이라이트 → 3개 선택지:
- **재도전** (리뷰 날짜 재설정)
- **내려놓기** (별 시들어 사라짐, 슬롯 해제)
- **그대로 두기** (상태 유지)

### 3.4 별자리 시스템

**진행 구조 (고정):** 북두칠성 → 큰곰자리 → 작은곰자리 → 자유 해금 (무료: 랜덤 / 프리미엄: 선택·생성 / 생일 별자리 옵션)

**북두칠성 (슬롯 7개 고정 — 첫 별자리 표준화):**

| 슬롯 | 한자 | 로마자 |
|---|---|---|
| 1 | 천추(天樞) | Dubhe |
| 2 | 천선(天璇) | Merak |
| 3 | 천기(天璣) | Phecda |
| 4 | 천권(天權) | Megrez |
| 5 | 옥형(玉衡) | Alioth |
| 6 | 개양(開陽) | Mizar |
| 7 | 요광(搖光) | Alkaid |

- 슬롯 이름 기본: 유저 이름 또는 AI 추천어 / 정보 아이콘 탭 시 원래 이름·의미 표시
- 북두칠성 완성 시 커뮤니티 인증 메시지 ("당신은 N번째 별지기입니다")

**플로팅 별:**
- 슬롯 모두 점유 시 새 후회는 플로팅으로 생성
- 슬롯 해제 시 가장 오래된 플로팅 별 자동 편입
- 3개 이상 누적 시 소프트 넛지 ("떠 있는 별이 3개 있어요. 혹시 다시 볼 시간이 됐을까요?")

### 3.6 별자리 아카이브 & 히스토리 타임머신

- **아카이브(무료)**: 별자리 완성 시 자동 저장 (별자리 이름, 완성 날짜, 전체 기간, 각 별의 후회·스토리·카테고리·날짜)
- **별하루 세상**(커뮤니티 현황): 총 별지기 수, 빛나는 별 총계, 완성된 별자리 수
- **히스토리 타임머신(v1.5 프리미엄)**: 완성된 별자리를 영화처럼 자동 재생. 오프닝 → 별자리별 재생 → 최종 요약

### 3.7 감정 위기 대응 프로토콜 (Launch-Blocking 해결)

- **v1.0**: 키워드 기반 로컬 감지 ("자살", "죽고 싶", "자해" 등 명시적 위기 표현만, 오탐 최소화)
- **조용한 개입**: 저장 정상 진행, 직후 1회만 부드러운 모달 노출, 닫으면 세션 내 재노출 안 함
- **안내 문구**: 자살예방상담전화 1393 (24시간, 무료)
- **v1.1+**: 키워드 + AI 맥락 판단 하이브리드
- **조건부 마일스톤**: 출시 후 3개월 내 정신건강 전문가 자문 → 문구 업데이트

### 3.8 계정 · 로그인

- **게스트 모드**: 설치 즉시 익명 임시 계정 (디바이스 ID 기반) / 북두칠성 완성까지 모든 기능 사용 가능
- 로그인 유도: 북두칠성 완성 축하 화면에서
- Firebase Auth anonymous-to-authenticated migration으로 기존 데이터 이관

**v1.0 로그인:** 애플 / 구글 / 이메일 매직 링크
**v1.1 추가:** 카카오 / 네이버

**프라이버시 설계:**
- 최소 스코프 (이메일·고유ID만, 이름·프로필사진·친구리스트 수집 X)

## 9. 리스크 관리 (PreMortem 요약)

### Launch-Blocking 6개 (모두 설계·계획 완료)
1. 감정 위기 대응 부재 → 키워드 감지 + 조용한 모달, 3개월 내 전문가 자문 (설계 완료)
2. 별자리 슬롯 불균형 → 북두칠성 고정 7개 표준화 (설계 완료)
3. 극복 스토리 필수 입력 이탈 유발 → 2단계 분리 방식 (설계 완료)
4. 플랫폼 미결정 → Flutter + Firebase 확정 (설계 완료)
5. 핵심 가설 미검증 → Phase 1 프로토타입 후 15~20명 지인 테스트 (계획됨)
